Creating A Page

In this section, we will learn about creating pages in doc-site.

The page component empowers you to create one-off standalone pages. You can use other fastn components, or markdown.

Adding global right-sidebar to my-ds module

If you want to show right-sidebar on all pages and the content of your right-sidebar is the same for all pages.

Just add sidebar property in ds.page component and pass its value as true.

Hereâ€™s what your ds.ftd module would look like:
